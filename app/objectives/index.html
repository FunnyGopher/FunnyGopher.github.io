I told you I do softwares

Objective I

Follow a software development process to analyze a problem, and to design, build, and test software solutions.

CrowdDJ

How Objective was met
CrowdDJ used a modified scrum process, using every two weeks as a sprint. Features to be completed were initially created for each sprint. After each sprint, there was a review session about what was and was not completed. Tasks for future sprints were then adjusted according to ahead or behind schedule the project was.

Partí

How Objective was met
Partí used a feature driven development process, where key features were initially designed by the team. Each team member was then given a feature to work on. As features were completed, they were reviewed, and added to the main build.

Team: Myself, Brandon Olson & Justin Fern

CrowdDJ

How Objective was met
CrowdDJ is a Java desktop application with an acompannying Android application. The desktop application was developed in Java 8 using JetBrains's IntelliJ IDE, while the Android application was developed using the Android SDK in Google's Android Studio.

How Objective was met
Fredrick's Quest for Hair is a desktop game written in Python, using the PyGame library for ease of development. The game was developed using IDLE.

Objective III

Kyles.Place

How Objective was met
Kyles.Place is a responsive website developed from scratch using HTML5 with Webcomponents, CSS3 and the Sass precompiler, jQuery Javascript libraries, and the Materialize styling library.

How Objective was met
BwaaahButton is a small Android mobile application developed with the Android SDK. The application was built to support Android 4.0 Ice Cream Sandwich, and targeted towards the major realease at the time, Android 5.0 Lollipop.

It's pretty sweet

CrowdDJ

A desktop and Android application that allow for groups of people to play their music like a Jukebox. Songs are loaded into the desktop application, and users can request songs via the Android application. Uses H2 Database for local storage, and embedded Jetty server for the mobile app to talk to, and JavaFX for the front end.

BwaaahButton

A small Android application that plays the dramatic bwaaah sound from the movie, "Inception." Also cycles through a couple random movie quotes every time the sound is played. Makes any situation sound like an epic movie. Use with caution.

Partí

An Android application that helps organize wicked parties. Hosts can create parties to invite people, and invitees and accept or decline invitations. The app also helps invitees with directions and such.

Uses H2 Database for local storage and a MySQL server with PHP to save events remotely.

Team: Myself, Brandon Olson & Justin Fern

Perspective Rectifier

This one's cool. Changes the perspective of an image using matrix calculation concepts we learned in Linear Algebra. And you thought you'd never need to learn higher level math.

Team: Myself, Brandon Olson & Nathan Reline

Scheduler

A scheduling application that would schedule activities for you, planning around things you're already doing.

It was also a failure in back end Java web service functionality. Many common back end tools were used for the first time including Apache Tomcat, H2 Database, Maven, and JavaFX, which we tried to use as a front end.

Team: Myself, Brandon Olson, Justin Fern & Nathan Reline
